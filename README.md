## 🔎 How search works

rawq combines two kinds of search:

### Lexical search
This finds exact words and file names. Use it when you know part of the name or code.

### Semantic search
This finds code that means the same thing, even if the words are different. Use it when you are not sure what the file is called.

### Combined search
rawq can mix both methods so you get useful results faster. This helps when you search a big project and do not know the exact term.
